Política de Privacidade

1. Das permissões de acesso

1.1 As permissões de acesso que são pedidas ao usuário são:

  • Acessar galeria;
  • Acessar câmera;
  • Ler/gravar no dispositivo.

A permissão de acesso à galeria é pedida para que o aplicativo acesse a galeria de fotos no dispositivo, assim, o usuário poderá selecionar uma foto para edição.

A permissão de acesso à câmera é pedida para que o usuário seja capaz de fazer uma foto através do aplicativo e editá-la em seguida.

Quanto a leitura e gravação no dispositivo, é para que o aplicativo seja capaz de ler as imagens e de salvar as edições no dispositivo do usuário.

1.2 Nenhuma das permissões serão utilizadas para qualquer outro fim que não seja a edição de foto quando o usuário utilizar o aplicativo.

2. Dos arquivos acessados no dispositivo

2.1 Nenhum dado é armazenado no aplicativo FilterApp ou em qualquer servidor. Não serão armazenadas nenhuma foto do usuário ou qualquer tipo de dado existente no dispositivo do usuário no aplicativo FilterApp ou em qualquer servidor.
O app NÃO coleta ou compartilha NENHUM tipo de dados do usuário.
